# Project-Showcase
Portfolio of selected course projects (Java) showcasing backend, data, and UI problem-solving.

## Featured Projects

### 1) WellingtonTrains (Java)
A query tool for Wellington train lines and timetables: station/line lookups, next services, and trip planning (including one-transfer trips).
- **Skills:** data parsing, collections (Map/Set/List), search logic, clean UI output
- **Highlights:** trip planning with transfer + route visualisation on map

### 2) DeShredder (Java)
Interactive tool to reconstruct shredded documents by dragging shreds into strips, reordering strips, and exporting the final image.
- **Skills:** GUI interaction, state management, algorithms
- **Highlights:** neighbour suggestion (pixel-edge difference) + save-to-image pipeline
